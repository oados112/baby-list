# 🍼 רשימת קניות ללידה

אתר קטן ופשוט לניהול משותף של רשימת קניות לתינוק — לכם ולבני/בנות הזוג.
שניכם עורכים מהטלפון, מסמנים מה כבר נקנה, מסמנים מה לוקחים מ**שני** (אחות
האישה) ומה אנחנו קונים, מחלקים לקטגוריות, מנהלים צ'ק־ליסט לתיק הלידה ועוקבים
אחרי התקציב. כל השינויים נשמרים אוטומטית ל-GitHub.

---

## הקמה — צעד אחר צעד

### 1. יצירת ה-repo
1. היכנסו ל-https://github.com והתחברו.
2. לחצו **New repository**.
3. שם: `baby-list` · בחרו **Public** · סמנו **Add a README** · לחצו **Create**.
4. העלו את הקבצים `index.html`, `styles.css`, `app.js`, `data.json` ל-repo
   (גרירה למסך ה-repo → **Commit changes**).

### 2. הפעלת האתר (GitHub Pages)
1. ב-repo: **Settings → Pages**.
2. תחת *Build and deployment* בחרו **Branch: main** ותיקייה **/(root)** → **Save**.
3. אחרי דקה תקבלו כתובת כמו:
   `https://<שם-המשתמש>.github.io/baby-list/` — זו כתובת האתר שלכם. 🎉

### 3. הפקת טוקן (כדי שהאתר ישמור לבד)
1. https://github.com/settings/tokens?type=beta → **Generate new token** (Fine-grained).
2. **Repository access** → *Only select repositories* → בחרו `baby-list`.
3. **Permissions** → *Repository permissions* → **Contents** → **Read and write**.
4. **Generate token** והעתיקו את הטוקן (מתחיל ב-`github_pat_...`).
   ⚠️ הטוקן מוצג פעם אחת בלבד — שמרו אותו זמנית.

### 4. התחברות באתר
1. פתחו את כתובת האתר בטלפון/מחשב → ייפתח מסך **הגדרות**.
2. מלאו: שם משתמש GitHub · `baby-list` · `main` · הטוקן → **שמירה והתחברות**.
3. חזרו על שלב 4 גם במכשיר של אשתך (ובכל מכשיר נוסף).
   💡 שמרו את הכתובת כסימנייה / "הוסף למסך הבית" בטלפון.

## איך משתמשים
- **קניות:** הוסיפו פריט (שם + קטגוריה + מקור + כמות). לכל פריט אפשר לסמן ✓ נקנה,
  לשנות כמות, להחליף מקור (אנחנו/שני בלחיצה על התג), להזין מחיר והערות.
- **סינון:** לפי קטגוריה, "רק מה שעוד לא נקנה", "רק מה שלוקחים משני".
- **תיק לידה:** צ'ק־ליסט נפרד למה לארוז לבית החולים.
- **תקציב:** סכומים אוטומטיים — סך הכל, "אנחנו" מול "שני", ונשאר עד היעד.

## סנכרון בין שניכם
- כל שינוי נשמר ל-GitHub אוטומטית (אינדיקטור "✓ נשמר" למעלה).
- כדי לראות שינויים של השני: האתר מתרענן לבד כל ~30 שניות וכשחוזרים לטאב,
  או לחצו על 🔄. (אין דחיפה חיה — צריך רגע לרענן.)

## אבטחה
הטוקן נשמר **רק בדפדפן של המכשיר** (localStorage) ולא נכתב לקוד. ה-repo ציבורי,
אבל רשימת קניות אינה מידע רגיש. אם הטוקן דולף — בטלו אותו ב-GitHub Settings
והפיקו חדש; הוא מוגבל ל-repo הזה בלבד.

## עריכת הקטגוריות / הוספת המון פריטים בבת אחת
אפשר לערוך את `data.json` ישירות ב-GitHub (פתחו את הקובץ → ✏️ → Commit). שלחו
לי את רשימת הקטגוריות והפריטים שלכם ואכניס אותם בבת אחת.
